## Build and Run

### 1. 开发环境

本项目采用微信小程序作为前端，Node.js + Express作为后端服务。

开发环境如下：

| 软件           | 版本要求  |
| -------------- | --------- |
| Windows 10/11  | 推荐      |
| Node.js        | v16及以上 |
| npm            | v8及以上  |
| 微信开发者工具 | 最新版    |

------

### 2. 项目构建

#### （1）获取项目源码

下载项目源码并解压至本地目录。

#### （2）安装后端依赖

进入后端目录：

```bash
cd server
```

安装项目依赖：

```bash
npm install
```

该命令将根据 package.json 自动安装项目所需依赖，包括：

- Express
- Axios
- Cors
- Body-Parser

等运行环境。

------

### 3. 启动后端服务

进入 server 目录：

```bash
cd server
```

执行：

```bash
node server.js
```

服务器启动成功后，即可为微信小程序提供接口服务。

------

### 4. 启动微信小程序

1. 打开微信开发者工具；
2. 选择“导入项目”；
3. 选择项目根目录 Programe01；
4. 填写测试 AppID 或开发者 AppID；
5. 点击“编译”。

编译成功后即可进入系统首页。

------

### 5. 运行流程

```text
安装依赖（npm install）
            ↓
启动后端（node server.js）
            ↓
打开微信开发者工具
            ↓
导入项目
            ↓
点击编译
            ↓
运行成功
```

------

### 6. 功能验证

系统启动后可正常使用以下功能：

- 用户登录
- 首页信息展示
- 天气查询
- AI宿舍调解
- 值日安排管理
- 宿舍账单管理
- 宿舍公约查看
- 个人中心管理

各模块能够正常进行数据交互与页面展示。
